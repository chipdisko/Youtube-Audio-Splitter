# YouTube Audio Splitter

## environment

python 3.12.4

## install packages

$ brew install ffmpeg
$ brew install yt-dlp
$ python3 -m venv ../Youtube-Audio-Splitter
$ source ../Youtube-Audio-Splitter/bin/activate
$ pip3 install -r requirements.txt

## pyinstaller

$ pyinstaller --add-binary 'recources;.' your_script.py

## Execute

$ python3 SPLYT.py
